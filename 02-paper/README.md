# 02 - 论文导读

本模块整理 LLM 后训练领域的经典与前沿论文，按主题分为三个方向。

## 子模块

| 目录 | 主题 | 核心问题 |
|------|------|---------|
| 2.1-SFT | 监督微调 | 如何用指令数据让模型遵循人类意图 |
| 2.2-RL | 强化学习对齐 | 如何用偏好信号进一步优化模型 |
| 2.3-Safety | 安全对齐 | 如何让模型拒绝有害请求、保持无害 |

## 阅读建议

1. 先读 InstructGPT 建立整体框架认知
2. 再读 SFT 系列，理解数据质量对效果的关键影响
3. 进入 RL 系列，对比 PPO / DPO / GRPO 的方法论差异
4. 最后读 Safety 系列，理解对齐税与安全的权衡

## 关键论文时间线

```
2022  InstructGPT (RLHF 奠基)
2023  Alpaca / Vicuna (低成本 SFT)
2023  DPO (去掉奖励模型)
2024  ORPO / SimPO (无参考模型偏好优化)
2024  DeepSeek-R1 (GRPO + 长链推理)
2025  RLVR 系列 (可验证奖励 + 规则信号)
```
